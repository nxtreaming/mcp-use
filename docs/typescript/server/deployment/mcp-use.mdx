Deploy your MCP server to **mcp-use cloud**, the recommended deployment platform for mcp-use servers. With a single command, you can deploy your server and get a production-ready URL.

## Prerequisites
**mcp-use Account**: Sign up at [mcp-use.com](https://mcp-use.com) or login via CLI

## Quick Deployment

Deploy your MCP server in one command:

<CodeGroup>
```bash npm
# Install the CLI (if you are using create-mcp-use-app this step is not needed)
npm install -g @mcp-use/cli

# Login to mcp-use cloud
npm run mcp-use login

# Deploy your server
npm run mcp-use deploy
```

```bash yarn
# Install the CLI (if you are using create-mcp-use-app this step is not needed)
yarn global add @mcp-use/cli

# Login to mcp-use cloud
yarn mcp-use login

# Deploy your server
yarn mcp-use deploy
```

```bash pnpm
# Install the CLI (if you are using create-mcp-use-app this step is not needed)
pnpm add -g @mcp-use/cli

# Login to mcp-use cloud
pnpm mcp-use login

# Deploy your server
pnpm mcp-use deploy
```
</CodeGroup>

That's it! Your server will be built and deployed automatically.

## GitHub Repository Deployment

If your project is a GitHub repository, `mcp-use deploy` will automatically detect it and offer to deploy from GitHub:

```bash
mcp-use deploy
```

You'll see:
```
GitHub repository detected:
  Repository: your-org/your-repo
  Branch:     main
  Commit:     abc1234

Deploy from GitHub repository your-org/your-repo? (y/n):
```

This approach:
- ‚úÖ Deploys directly from your GitHub repository
- ‚úÖ Automatically detects build and start commands
- ‚úÖ Uses the latest commit from your branch
- ‚úÖ No need to upload source code manually
